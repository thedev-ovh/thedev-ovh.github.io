Create Your Free Domain

Follow these simple steps to get your {name}.thedev.ovh

  1. Step 1: Fork the thedev.ovh repository.
  2. Step 2: Create a new file in the domains/ folder with the name of your subdomain (e.g., {yourname}.thedev.ovh.json).
  3. Step 3: Add the following content to your file: 
    {
  "subdomain": "yourname",
  "owner": {
    "email": "your-email@example.com",
    "github": "your-github-username"
  },
  "record": {
    "CNAME": "yourwebsite.github.io"
  },
  "proxied": false
}
    Replace yourname, your-email@example.com, and yourwebsite.github.io with your information.
  4. Step 4: Create a pull request with your changes. The team will review and approve your request.
  5. Step 5: Once approved, your domain yourname.thedev.ovh will be live!
